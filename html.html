Sintaxe

Introdução

O HTML é ultilizado como uma linguagem de marcação para desenvolver páginas e documentos para a internet.

Sua sigla significa hypertext markup language, expressão inglesa que traduzindo fica linguagem de marcação de hipertexto.

Parágrafo

Parágrafos são definidos através das tag < p > e < /p >.

Exemplo:

< p > assim se define um parágrafo < /p >

Quebra de linha

Quebra de linha é definido através de uma só tag < /br >.

Exemplo:

assim se define uma quebra de linha < /br >

Link

O link é definido através da tag < a > < /a > e possui o atributo href="" para definir o destino.

Exemplo:

< a href="destino.html"> destino para o interno < /a >.

< a href="https://www.facebook.com/"> destino para um link < /a >.

Imagem

A imagem é definido atraves da tag < img/ > não possui tag de fechamento, possui os atributo href="destino" e alt="descriçao da imagem".

Exemplo:

< img href="campus.jpeg" alt="imagem aéria" /> Destino interno.

< img href="https://images.app.goo" alt="imagem de um cavalo"/> Destino via link.

Títulos de Seção

O título de seção é definido pela tag < h1 > < /h1 > indo até o nível 6.

Exemplo:

< h1 > título de nível 1 < /h1 >

< h2 > título de nível 2 < /h2 >

< h3 > título de nível 3 < /h3 >
